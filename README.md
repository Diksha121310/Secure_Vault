# 🔐 SecureVault

SecureVault is an enterprise-inspired cybersecurity application designed to securely store passwords, sensitive notes, API keys, digital assets, and encrypted files while providing users with actionable security insights.

The project combines modern backend architecture with a responsive frontend to simulate a production-ready password manager and secure vault system.

---

## ✨ Features

### 🔑 Authentication & User Security

* JWT-based Authentication
* Secure User Registration & Login
* Password Hashing using BCrypt
* Forgot Password with Email OTP Verification
* Password Reset Workflow
* Change Password
* Login Rate Limiting
* Protected Routes
* Automatic Session Expiration

---

### 🔒 Secure Vault

Store encrypted:

* Passwords
* Secure Notes
* API Keys
* Digital Assets

Features include:

* AES Encryption
* Add Items
* Edit Items
* Delete Items
* Categorized Vault
* Search & Organization
